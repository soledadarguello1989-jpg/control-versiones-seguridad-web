# Estrategia de control de versiones y seguridad - Sistema de Biblioteca

Este repositorio contiene la estrategia de control de versiones y seguridad aplicada al archivo `Login.java` del Sistema de Biblioteca.

El trabajo se basa en la plantilla de control de versiones entregada en la asignatura, respetando la información original registrada en el historial de versiones, registro de cambios, conflictos y recuperación de versiones.

## Objetivo

Diseñar una estrategia de control de versiones y cambios mediante GitHub, considerando ambientes de Desarrollo, Certificación y Producción, e incorporando pruebas de seguridad antes del lanzamiento del producto final.
